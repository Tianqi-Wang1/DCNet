A demo implementation of the CIFAR100-Split10 experiment described in the submission, entitled **On the Discrimination and Consistency for Exemplar-Free Class Incremental Learning**.

**Installation & Requirements**

The current version of the codes has been tested with Python 3.8.19 on both Windows and Linux operating systems with the following requirements:

- 1x RTX 3080
- cuda==12.7
- numpy==1.24.3
- scipy==1.10.1
- scikit-learn==1.3.2
- torch==2.0.0
- torchvision==0.15.0
- tensorboardx==2.6.2.2
- diffdist==0.1

Please install the necessary packages

**How To Use**

Please run:

`sh train_cifar100_10t.sh`
